# Give the values of `i` and `j` after each of the following expression statement has been executed. (Assume that `i` has the value `1` initially and `j` has the value `2`)

- a) `i += j;`
- b) `i--;`
- c) `i * j / i;`
- d) `i % ++j;`

<details>
<summary>Show Answer</summary>

- a) `3 2`
- b) `2 2`
- c) `2 2`
- d) `2 3`

</details>
