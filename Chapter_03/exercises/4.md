# Suppose that we call `scanf` as follows:

```c
scanf("%d%f%d", &i, &x, &j);
```

## If the user enters

```txt
10.3 5 6
```

## What will be the values of `i`, `x` and `j` after the call? (Assume `i` and `j` are `int` variables and `x` is a `float` variable)

<details>
<summary>Show Answer</summary>

- `i` &mdash; 10
- `x` &mdash; 0.3
- `j` &mdash; 5

</details>
